# 贪吃蛇 🐍

经典贪吃蛇游戏，基于 Flutter 开发，支持 Android 和 Windows 双平台。

## 功能

- 🎮 经典贪吃蛇玩法：移动、吃食物、增长、碰撞检测
- 🎯 三种难度：慢速 / 中速 / 快速
- 📐 三种棋盘大小：15×15 / 20×20 / 25×25
- 🏆 最高分记录持久化
- ⏯️ 暂停 / 继续 / 重新开始
- ⌨️ 键盘方向键 + 触摸方向键控制
- 🌓 深色 / 浅色主题切换
- 📱 自适应布局（手机 / 折叠屏 / 平板 / 桌面）
- 🎵 音效反馈：吃食物、游戏结束音效
- 🏅 排行榜：多条目最高分记录
- 🌐 简体中文界面

## 开发

### 环境要求

- Flutter 3.44.1+
- Android SDK 34+
- Windows 10+

### 运行

```bash
flutter pub get
flutter run
```

### 测试

```bash
flutter test
```

## 技术栈

- Flutter 3.44.1
- Provider 状态管理
- Material 3 设计
- SharedPreferences 持久化
- window_manager 桌面窗口管理
